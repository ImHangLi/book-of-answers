# The Book of Answers

A mystical, atmospheric web experience inspired by the classic Book of Answers. Type your question, press Enter, and receive guidance from the universe.

**[Live Demo](https://book-of-answers-three.vercel.app)**

## Features

- Type a question and receive a random answer from ~270 curated responses
- Canvas-based floating gold particle field
- Blur-in reveal animation with ornamental typography
- Full keyboard accessibility (Space/Enter navigation)
- `prefers-reduced-motion` support
- Mobile responsive with safe area insets

## Tech Stack

- **React 19** + **TypeScript**
- **Vite** for build tooling
- **Framer Motion** for UI transitions
- **Canvas API** for particle effects and background
- **pnpm** as package manager

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
